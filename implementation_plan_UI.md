# UIデザイン・操作性のプレミアム改修プラン

アプリ全体の見た目（UI）と操作性（UX）を大幅に向上させ、先進的でプレミアムな**ブルー系の透明感（グラスモルフィズム）**デザインに統一します。モバイルでの操作性（指先サイズ、余白の最適化）にも徹底的にこだわります。

---

## 🎨 デザインコンセプト

1. **背景と透明感（Glassmorphism）**
   - 背景には、微細なグラデーション（`bg-gradient-to-tr from-blue-50 via-indigo-50/50 to-sky-100/50`）を採用します。
   - カード要素には、すりガラスのような質感（`bg-white/70 backdrop-blur-md border border-white/80 shadow-xl shadow-blue-900/5 rounded-3xl`）を採用し、圧倒的な高級感を演出します。
2. **情報の視認性（Visual Hierarchy & Grouping）**
   - 文字と背景のコントラストを高め、情報の優先度（見出し、説明、メタデータ）をサイズとウエイトで明確に区別します。
   - セクションごとにゆとりのある余白をとり、カード形式で綺麗にグループ化します。
3. **モバイル対応ボタン（Touch Targets）**
   - すべてのボタンは丸みを帯びた形状（`rounded-2xl` または `rounded-full`）にし、高さはタッチしやすい指先サイズ（最小 `52px` 〜 `56px`）に統一します。
   - ボタンを押した瞬間の微細なインタラクション（`active:scale-[0.98] transition-all`）を付与し、操作の心地よさを実現します。

## 🧪 検証プラン

### 手動検証（モバイルシミュレータ）
1. モバイル画面において、ボタンの高さが指先で十分に押しやすいサイズ（52px以上）になっているか確認。
2. 興味レベルボタンを切り替えたときのトランジション効果が滑らかか確認。
3. すりガラスのような透明感（グラスモルフィズム）が、背景のグラデーションに対して美しく映えているか確認。
4. 情報のグループ化が崩れておらず、十分な余白（`space-y-6` や `gap-4` 等）があるか確認。
